# Crypter-on-C
A simple Crypter written in C

## Features
— unhook ntdll
— AES-GCM decryption of payload
— process hollowing in svchost.exe
— manual relocation of PE
— rewriting PEB
— substitution of entry point

## Some weaknesses
— The path is written too rigidly(C:\\Windows\\System32\\svchost.exe)
— Fixed seed
